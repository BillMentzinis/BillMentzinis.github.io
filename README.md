# billmentzinis.github.io

Personal website of Vasileios (Bill) Mentzinis, hosted at [billmentzinis.github.io](https://billmentzinis.github.io).

Built with [Astro](https://astro.build) and the [Astrofy](https://github.com/manuelernestog/astrofy) template.

## Local Development

```bash
npm install
npm run dev      # http://localhost:4321
```

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically.
